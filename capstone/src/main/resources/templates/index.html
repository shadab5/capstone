SUPPLIER FINANCE MANAGEMENT SYSTEM

Supplier Finance System is a Web Application using Microservice architecture with multiple microservices communicating with each other, where it allows suppliers and clients to register/add/view the invoices from system. Supplier finance is a concept, describing a set of technology-based solutions that aim to lower financing costs and improve business efficiency for buyers (or clients) and sellers (or suppliers) linked in a sales transaction. The idea work by automating and tracking invoice approval and settlement processes, from initiation to completion.
